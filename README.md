# KV Monitor (Portable)

Lightweight monitoring panel for Knowledge Vault (DMR system).

## Features
- System health monitoring
- API endpoint checks
- Lightweight UI

## Requirements
- PHP 8.x
- Web server (Nginx or Apache)

## Usage
Place folder anywhere and serve via web server:

Example:
http://localhost/kv-monitor-portable/

## Configuration
Edit:
config.php

## Notes
- No framework dependencies
- Fully portable
- Designed for internal infrastructure monitoring
